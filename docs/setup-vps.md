# Заказ VPS в Aeza для office-monitoring

## Параметры

- Хостер: **Aeza** (тот же, где qc-zvonki — https://aeza.net)
- Локация: **Стокгольм**
- ОС: **Ubuntu 24.04 LTS**
- Тариф: **8 vCPU / 16 GB RAM / 240 GB NVMe** (~3000-4000 ₽/мес)
- Тип: VPS / VDS (не Cloud, чтобы цена была фиксирована)

## Шаги в личном кабинете Aeza

1. Войти на https://my.aeza.net/
2. «Купить сервер» → выбрать локацию **Стокгольм**
3. Выбрать тариф с 8 vCPU / 16 GB RAM (категория «Премиум» или аналог)
4. ОС: **Ubuntu 24.04**
5. Пароль root: сгенерировать новый, сохранить в менеджере паролей
6. Период оплаты: на старте — 1 месяц (потом перейдём на годовую если ОК)
7. Оплатить

## После создания

Прислать:
- IP-адрес нового сервера
- Пароль root (или способ передачи — лучше через KeePass/Bitwarden/1Password share)

Тогда я подключусь и поставлю Python + FastAPI + Caddy + домен/HTTPS.
