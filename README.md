# office-monitoring

Система мониторинга работы менеджеров в офисной сети (~90 офисов × 2 чел = ~180 эндпоинтов).

## Что делает

1. **Анализ встреч** — запись микрофона ноута во время очных консультаций с клиентом, транскрипция через Whisper, LLM-оценка качества продажи.
2. **Анализ звонков** — переиспользует [qc-zvonki](https://github.com/dogmat1910-tech/qc-zvonki) (Скорозвон) + добавит ЮИС.
3. **Анализ активности между встречами** — трекинг активного окна и времени в приложениях/на сайтах. Категоризация рабочее/личное/бездействие.
4. **Сводка по дню** — таймлайн на каждого менеджера: что делал, сколько времени, рабочее vs личное.

## Структура

```
office-monitoring/
├── agent/       # Windows-агент (Python + PyInstaller + NSSM)
├── server/      # FastAPI: приём данных, очередь обработки, Whisper, LLM
├── dashboard/   # Веб-дашборд: встречи, звонки, таймлайн дня
└── docs/        # Документация и заметки
```

## Юридическая база

- Корп. ноутбуки, согласие на обработку ПДн подписано, в трудовых договорах есть пункт о мониторинге.
- Видимая индикация записи (иконка в трее) — обязательно по 152-ФЗ.
